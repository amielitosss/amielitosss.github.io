Langages Frontend

HTML (HyperText Markup Language) est le langage standard pour créer la structure et le contenu des pages Web.

CSS (Cascading Style Sheets) est utilisé pour styliser et concevoir visuellement des pages Web, y compris les couleurs, les mises en page et les polices.

JavaScript est un langage de programmation qui ajoute de l'interactivité et un comportement dynamique aux sites Web, comme des animations et des interactions avec les utilisateurs.

Bootstrap est un framework front-end populaire qui vous aide à créer rapidement des sites Web réactifs et adaptés aux mobiles à l'aide de composants CSS et JavaScript prédéfinis.
